
Acerca del programa:

Este código es una solución mediante el método de portero en el comedor al problema de los filósofos comensales. 
Esta solución se implementa basandose en los siguientes requisitos:
    . Son 5 filósofos.
    . El programa se acabara cuando cada uno de los filósofos haya comido 6 veces.
    . En pantalla se debe mostrar cuando un filósofo se encuentra pensando, comiendo, cuando termina de comer y el
      número de comidas que lleva. 

Requisitos del sistema para compilar:

Para poder correr este programa se necesita un sistema operativo windows de 64bits
y tener instalado la versión para 64 bits de g++. 

Notas:

    . Si se desea agregar más filósofos, cambiar el valor con el que se inicializa el semáforo comedor y asignarle 
      un valor de n-1 donde n representa la nueva cantidad de filósofos
